Branded fundraising deck (20 slides) for **GO FETCH FIDO** using Reveal.js.

## Run the deck locally

```bash
npm install
npm start
```

Then open `http://127.0.0.1:8000`.

## Export PDF

```bash
npm run export:pdf
```

Output: `dist/GO-FETCH-FIDO-angel-deck.pdf`

## Customize branding

- Edit `index.html` and replace `REPLACE_ME` with your phone number + website.
- Colors/theme live in `styles/gofetchfido-theme.css`.
- Replace the silhouette in `assets/running-dog-silhouette.svg` with your real logo/art.
